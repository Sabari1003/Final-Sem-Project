Enhancing Machine Learning Predictions with GAN-Generated Synthetic Data

Project Overview

This project focuses on improving machine learning predictions using GAN-generated synthetic data. The system generates realistic synthetic datasets to address data scarcity, class imbalance, and privacy concerns while maintaining predictive performance.

Features

* Upload and preprocess datasets
* Generate synthetic data using CTGAN/GAN
* Compare real and synthetic datasets using EDA
* Train ML models using synthetic data
* Analyze bias and privacy
* Generate reports and visualizations

Tech Stack

* Python
* Flask
* CTGAN / GAN
* Pandas
* NumPy
* Scikit-learn
* HTML
* Bootstrap

Team Size

4 Members

How to Run

1. Clone the repository.
2. Install the required dependencies:
   pip install -r requirements.txt
3. Run the Flask application:
   python app.py
4. Open the application in your browser.

Project Outcome

The project demonstrated that GAN-generated synthetic data can improve machine learning model performance while preserving data privacy.
